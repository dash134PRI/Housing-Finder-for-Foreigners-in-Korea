# HomeBridge Korea

A hackathon MVP for a foreigner-friendly housing decision-support assistant in Korea.

## What it does
- Provides persona-based housing recommendations
- Uses transparent rule-based scoring
- Shows pre-generated AI explanations
- Flags things to double-check before contacting an agent
- Extracts structured listing signals and turns them into explainable guidance
- Generates Korean messages from verified signal-based templates
- Works as a frontend-only demo using bundled JSON data
