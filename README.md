# HomeBridge Korea

A hackathon MVP for a foreigner-friendly housing decision-support assistant in Korea.

## What it does
- Provides persona-based housing recommendations
- Uses transparent rule-based scoring
- Shows pre-generated AI explanations
- Flags things to double-check before contacting an agent
- Extracts structured listing signals and turns them into explainable guidance
- Generates Korean messages from verified signal-based templates
- Works as a frontend-only demo using bundled JSON data

## Demo script
Many listings in Korea hide real costs in maintenance and heating. Our system estimates the true monthly burden and warns users before they make a mistake.

We do not ask AI to guess. We extract structured housing signals first, then generate explainable guidance and Korean messages from verified templates.

## Tech stack
- Next.js
- React
- TypeScript
- Static JSON dataset

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3002

## Deploy
Deploy directly on Vercel:
1. Push this repo to GitHub
2. Import the repo in Vercel
3. Select Next.js preset
4. Deploy

No backend or API keys are required for the MVP.
